Welcome to Jacob Kodner's Exhibition Website!

This website you are looking at right now was made from scratch. This means that every thing you see about this website, every letter of text, and visual effect were made by typing out lines of code. The languages that have been used to make this website are HTML, CSS, Javascript, and Python. All of these languages have contributed to the way this website appears and functions.

The reason why I chose to do a website for my Exhibition Project is because it introduces a special type of programmer, a Web Developer, and what they do to fulfill their job. A Web Developer manages and maintains websites, to appeal the clientele and users. They maintain site layouts and functions, along with making the site user-friendly. All of these facts I have presented to you about Web Developers are present in my Exhibition Paper. I understand that Web Development was not my main focus for my Exhibition, but I wanted to do this to provide a visual understanding of the functon of programmers..

This website has been made to entertain users, and show what a Web Developer can offer. I have made a replica of the famous Cookie Clicker. My version and the original Cookie Clicker are quite similar in concept, but differ in page layout and color scheme. In addition, I have included a simple drawing program, to show that programmers can provide programs that can be entirely customizable and artistic. I have also added a game called Find the Invisible Pixel, which was inspired by a game called Find the Invisible Cow (similar concept, yet different layout and gameplay). Warning: It is quite difficult!

As you can see, being a Web Developer can be quite fun. Let your imagination do the work! Have Fun!

An Brief Introduction To Web Development

This website was programmed in 4 different languages: HTML, CSS, Javascript, and Python. I thought it would be proper to give an introduction to what these languages do.

The best way to explain what these languages do, I will use an anology. I will compare the structure of a website to a human body.

To start off, I would like to explain HTML. HTML stands for Hypertext Markup Language. I like to think HTML as the bones that make up a human body. It adds structure. HTML is the main language that creates web pages and information that can be displayed in a web browser.

The next topic I am explaining is Javascript. I like to think Javascript as the muscle that keep the bones moving. It is commonly used with HTML to make web pages more interactive and dynamic, rather than static.

Next is CSS, which stands for Cascading Style Sheets. CSS is like the skin that covers it all, and makes the structure unique. CSS describes the looks and formats of the webbpage. It adds color, position, borders, padding, and much more.

Finally, Python is the one that gives life to the body. It is like the nervous system and brain. It works with the server of the website. A server is a system that responds to requests across a computer network. It "serves" out the code for each webpage.

As you can see, all of these languages are necessary for creating a webpage, and work together to fulfill their functions. There are many more languages that can help with server-side and web-page interaction such as PHP, Ruby, Java, Ruby, Perl, Flash, SQL, and much more!

I hope you have learned something about Web Development today, and maybe you might want to learn it!
